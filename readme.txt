== Description ==

Document hierarchy pages like a book and provide easy navigation between pages.


[Documentation](https://celtislab.net/en/wp-plugin-hpgnav/ )


== Used library ==

jquery.mjs.nestedSortable.js  https://github.com/ilikenwf/nestedSortableã€€ MIT license


== Installation ==

1. Upload the `hierarchy-pages-nav` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open the setting menu, it is possible to set the execution condition.
